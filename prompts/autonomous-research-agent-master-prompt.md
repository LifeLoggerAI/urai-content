# Autonomous Research Agent Master Prompt

A reusable, audit-ready research protocol for autonomous agents. It defines how an agent should scope a question, prioritize sources, search and screen evidence, extract structured data, synthesize findings, handle uncertainty, verify claims, and package final deliverables.
